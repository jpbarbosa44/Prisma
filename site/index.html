Por que o Prisma

Tudo o que importa, sem a parte chata.

Nada de planilha gigante nem app que pede seus dados bancários. O Prisma é direto: abre rápido, funciona offline e cabe num único arquivo que é só seu.

🔒

Privacidade total

Roda 100% local. O modo web sobe só em 127.0.0.1 — nada sai da sua máquina, nunca.

⚡

Rápido e leve

Um binário estático, sem dependências. Abre em tela cheia como o btop e responde na hora.

📦

Um único arquivo

Todos os seus dados num SQLite só. Faça backup copiando um arquivo. Leve para onde quiser.

Offline por padrão

Seus dados são seus. Ponto.

Sem cadastro, sem nuvem, sem telemetria. O Prisma guarda tudo num arquivo SQLite local — você é o único dono.

  • ✓Funciona 100% sem internet, em qualquer lugar.
  • ✓Backup é copiar um arquivo. Migração também.
  • ✓O modo web escuta só em 127.0.0.1 — nada exposto na rede.
  • ✓Código aberto: você pode auditar tudo.

Como usar

Do zero ao primeiro lançamento.

Três passos para começar — e a partir daí tudo é teclado. Os mesmos comandos servem na interface e em scripts. Para o guia completo, com todas as telas, filtros e receitas, veja o manual.

1

Instale

Baixe o binário do seu sistema e coloque-o no PATH. É um arquivo único, sem dependências — nada mais para instalar.

2

Abra

Digite prisma e a interface abre em tela cheia. Prefere o mouse? prisma --web abre no navegador.

3

Lance

Cadastre uma conta, registre o que entra e sai, e veja seu saldo consolidado na hora.

primeiros passos
# cadastrar uma conta bancária e uma carteira $ prisma conta add --nome "Nubank" --tipo corrente --saldo 1.500,00 $ prisma carteira add --nome "Dinheiro" --saldo 200 # ver a posição geral consolidada $ prisma saldo $ _
# uma despesa simples e uma receita $ prisma pagar add --desc "Aluguel" --valor 1200 --venc 05/07 --cat moradia $ prisma receber add --desc "Freela" --valor 800 --conta 1 # parcelar (divide o TOTAL) e listar os pendentes $ prisma pagar add --desc "Notebook" --valor 3.600,00 --parcelas 10 $ prisma lancamentos --pendentes # marcar como pago/recebido $ prisma quitar 4 $ _
# cadastrar o cartão (fechamento e vencimento) $ prisma cartao add --nome "Nubank" --fechamento 20 --vencimento 27 --conta 1 # compra parcelada cai na fatura do ciclo $ prisma pagar add --desc "Tênis" --valor 400 --parcelas 4 --cartao 1 # ver a fatura aberta e pagá-la em bloco $ prisma fatura --cartao 1 $ prisma fatura pagar --cartao 1 $ _
# salário e aluguel: lançam sozinhos todo mês $ prisma recorrencia add --tipo receber --desc "Salário" --valor 5000 --dia 1 $ prisma recorrencia add --tipo pagar --desc "Aluguel" --valor 1300 --dia 10 # assinaturas: recorrência de despesa com total mensal $ prisma assinaturas add --desc "Netflix" --valor 39,90 --dia 20 --cartao 1 $ prisma assinaturas listar $ _
# grupo de 2 pessoas: o Prisma conta só a SUA parte $ prisma grupo add --nome "Eu e a Maria" --pessoas "Eu, Maria" # despesa de R$ 300 pesa só R$ 150 no seu bolso $ prisma pagar add --desc "Mercado" --valor 300 --grupo 1 # total cheio, sua parte e gasto do mês $ prisma grupo listar $ _

Três formas de usar

Terminal, comandos ou navegador.

A mesma engine, do seu jeito. Navegue com o teclado na interface de tela cheia, automatize com comandos diretos, ou abra no browser quando preferir o mouse.

  • ▸Interface TUI — tela cheia, navegação com ↑/↓ e atalhos.
  • ▸Comandos diretos — prisma pagar add ... para scripts.
  • ▸Modo web — prisma --web abre as mesmas telas no navegador.
comandos diretos
# cadastrar uma conta $ prisma conta add --nome "Nubank" --saldo 1.500,00 # lançar uma despesa $ prisma pagar add --desc "Mercado" --valor 240 # abrir no navegador (só em 127.0.0.1) $ prisma --web --porta 7747 → http://127.0.0.1:7747 $ _
